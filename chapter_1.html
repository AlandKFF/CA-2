Chapter 1: Computer Architecture - Concepts Summary

1. What is a Computer? & Key Components

Question Answer
What is a Computer? A programmable machine that performs input/output and processes data.
Key Components
  • Computation: Processing data
  • Communication: Data transfer (I/O)
  • Storage: Memory

2. Computer Must/Generally Have

Category Components
Must Have
  • CPU
  • I/O Device
  • Memory
Generally Have
  • System Bus
  • Cache Memory
  • I/O Bus
  • Memory Controller

3. Major Classes of Systems

System Class Description
Traditional PC High performance systems
Notebook Portability, size, and energy constraints
Smart Phone Battery and cost constrained
Data Center Ultra-scale, cost constrained

4. Computer Components

Component Type Description
Computer Hardware Physical parts responsible for data I/O during task execution.
Computer Software Programs that control, integrate, and manage the hardware.

5. Computer Software Parts

Type Description
Application Software Used to accomplish specific tasks; may consist of a single program.
System Software Files and programs that make up the computer's operating system (OS).

6. Computer System Components

Component Description
Hardware Provides the basic computing resources.
Operating System (OS) Controls and coordinates the use of hardware resources.
Application Programs Utilize hardware and OS resources to solve computing problems.
User Human or software agents who use the computer.

7. High-Level vs. Low-Level Languages

Language Level Description
High-Level Close to human language; examples include Java.
Low-Level Closer to machine language; examples include Assembly.

8. Abstraction

Concept Description
Abstraction Allows a higher level to only know about the interface to the lower level, not how it is implemented.
Improves productivity by hiding implementation details.

9. Computer Architecture (CA) and Its Attributes

Aspect Description
Computer Architecture (CA) The art and science of designing, selecting, and interconnecting computer hardware components and designing the hardware/software interface to meet specific goals.
Attributes
  • Instruction set
  • Data representation
  • I/O mechanism
  • Addressing techniques

10. Organization and Its Attributes

Concept Description
Computer Organization The way in which computer hardware components are connected and operate together to form a computer system.
Attributes
  • Control signals
  • Interface between computer and peripherals
  • Memory technology

11. Family Concept

Concept Description
Family Concept All computers within a family share the same architecture, ensuring compatibility and consistency.

12. Hierarchic System

Concept Description
Hierarchic System A set of interrelated subsystems that is essential for both the design and description of a computer system.

13. Structure

Aspect Description
Top-Level Structure Key components include:
  • CPU
  • Memory
  • I/O Devices
  • System Interconnection
CPU Structure Composed of:
  • Control Unit
  • Registers
  • ALU (Arithmetic/Logic Unit)
  • Interconnection mechanisms
Control Unit Responsible for issuing control signals; includes:
  • Control unit registers and decoders
  • Control memory
  • Sequencing logic

14. Function & Basic Computer Operations

Aspect Description
Function The operation of individual components as part of the overall structure.
Basic Computer Functions
  • Data Processing
  • Data Movement
  • Data Storage
  • Control
Operations
  • Data Movement: Transferring data between peripherals/communication lines (e.g., keyboard to screen).
  • Storage: Reading/writing data between external environment and storage.
  • Processing from/to Storage: Example: updating a bank account.
  • Processing from Storage to I/O: Example: printing a document.
